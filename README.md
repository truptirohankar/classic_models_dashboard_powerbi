# Classic Models Dashboard Power BI

Overview :
The Classic Model Dashboard is a Power BI project built to analyze sales, customers, products, and performance for the Classic Models dataset. The dashboard provides insights into sales data, product lines, customer countries, and net profit analysis.

Power Bi Visualization :
-Sales Analysis: Sales by product line, Sales by Country
-Order Insigh : Order by Product line, Order by year
-Product Dashboard: Top 5 selling Product, Bottom 5 Selling Product
-KPI Tracking: Total Sales, Profit margins, Top Selling Product

Dataset :
Used multiple CSV file
